This page shows what this small library SSChordsLib can do

When I developed a website to store my favourit songs with chords, I wanted a library that I could easily enter chords and transpose easily. At the time I could not find any, so I developed a small Javascript library using my limited knowledge in music and vast knowledge in programming. The result is this library. This will make it easier for you to enter chords, format and allow your viewers to transpose.

        // How to use the library in your page
        <script type="text/javascript" src="/SSChordsLib.min.js"></script>
        <script type="text/javascript"
            // create an instance of the class
            var lib = new SSChordsLib();
        
            // To format      
            lib.format("unformatted chords text"); // from a textarea etc

            // To unformat (you may want to go back to row text to make changes)
            lib.unformat("formatted chords HTML"); // from a div etc.

            // To transpose (plus). (Here you will bee passing in an HTML DOM element wich has the formatted code)
            lib.transpose(document.body.querySelector(".editorText"), 1);
 
            // To transpose (minus). (Here you will bee passing in an HTML DOM element (div) wich has the formatted code)
            lib.transpose(document.body.querySelector(".editorText"), -1);
       
        </script>

What you type in

  • All chords must start with &
  • Supported chords are: "Ab", "A", "Bb", "B", "C", "C#", "D", "Eb", "E", "F", "F#", "G"
  • Start Intro/Inter with [[ and finish with ]]
  • Female parts start with f{{ and finish with }}
  • Male parts start with m{{ and finish with }}
